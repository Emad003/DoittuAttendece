PREVIOUS FEATURES
=================

MULTI-SUBJECT SUPPORT FOR FACULTY
---------------------------------
Allow a faculty member to teach multiple subjects (e.g., C Language and C Language Lab)
and select which subject they're managing when using the dashboard.

GOOGLE SHEETS SETUP
-------------------
Update your faculty sheet with comma-separated subjects:

| Name     | Email              | Password | Subjects                      |
|----------|--------------------|----------|-------------------------------|
| John Doe | john@doit.edu      | pass123  | C Language, C Language Lab    |
| Jane Doe | jane@doit.edu      | pass456  | Data Structures, OOP Lab     |

Note: The column name should be "Subjects" (not "Subject")


HOW IT WORKS
------------
1. Faculty logs in with email/password
2. admin-login.html converts "C Language, C Language Lab" to an array
   ["C Language", "C Language Lab"]
3. In dashboard.html sidebar, a dropdown appears to select subject
4. Faculty selects which subject to manage
5. All attendance data, reports, and alerts are filtered by selected subject


FILES MODIFIED

2. dashboard.html

USAGE
-----
1. Update Google Sheets with comma-separated subjects
2. Faculty logs in
3. Select subject from dropdown in sidebar
4. View/take attendance for selected subject
5. Switch subjects anytime using the dropdown


NOTES
-----

- Attendance for "C Language" and "C Language Lab" are stored separately
- Reports can be generated for each subject individually
- Alerts are sent for the currently selected subject
